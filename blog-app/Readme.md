### backend development 
1. create git repo :
    git init
2. add .gitignore files
3. create .env file for environment variables and read data from .env with "dotenv" module
4. generate package.json
5. create express app
6. connect to a database
7. add middlewares(bodyparsing,error handling)
8. design schemas and create models
9. design restapi for all resources and connect it in server.js
9. create a folder services
